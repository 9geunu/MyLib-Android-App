# MyLib-Android-App

Kotlin으로 구현한 나의 서재 앱입니다.

## Architecture

MVVM 구조로 설계되었습니다. 

Room, LiveData, Flow, ViewModel, Coroutine, Retrofit, Glide 등을 사용했습니다.

## Features

신간 도서 / 도서 검색 / 북마크 / 도서 정보 확인 / 메모 기능을 구현했습니다.
